Project Overview

Project Name

Online Liquor Licensing Portal (OnlineLiquorPortal)

Project Sponsor

SAB (South African Breweries) - Full Project Funding

Beneficiary & Regulatory Authority

KZNERA - KwaZulu-Natal Liquor Authority responsible for liquor licensing in KZN

Project Objective

Develop a customer-facing online portal that integrates with KZNERA's internal eLMS system for public liquor license applications

Current System Landscape

eLMS (Existing Liquor Management System)

  • Internal system used by KZNERA staff
  • Processes liquor licensing as per the Act
  • Not customer-facing
  • Developed and maintained by IT Related
  • Backend processing and workflow management

The Need for OnlineLiquorPortal

Gap Identified: The public currently has no online access to submit liquor license applications. All applications must be submitted manually, leading to inefficiencies and service delivery challenges.

Solution: Create a customer-facing portal that allows the public to submit applications online, which will integrate with the existing eLMS through APIs.

Project Challenges & Bottlenecks

Primary Issue: IT Related team is experiencing significant delays in developing and delivering the API endpoints required to integrate the OnlineLiquorPortal with the existing eLMS system.

API Development Issues

  • Delayed API endpoint delivery
  • Incomplete API documentation

  • Resource constraints at IT Related
  • Testing environment unavailability

Integration Challenges

Impact of API Bottlenecks

Timeline Impact

The API development delays have created a critical path blockage. Frontend development is 80% complete but cannot proceed with integration testing. This creates a cascading delay affecting all subsequent phases.

Project Dependencies

  • Blocked: Integration testing - waiting for API completion
  • Blocked: End-to-end testing - requires working integration
  • Blocked: User Acceptance Testing - needs functional system
  • At Risk: Training and documentation - dependent on final system
  • At Risk: Deployment timeline - no confirmed date possible

Stakeholder Impact

  • SAB: Uncertainty around return on investment timeline
  • KZNERA: Continued reliance on manual processes
  • Public: Delayed access to improved licensing services
  • Development Teams: Resource inefficiency and morale concerns
